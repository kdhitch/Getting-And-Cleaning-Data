Getting and Cleaning Data Course Project CodeBook
---

This file describes accelerometer and gyroscope data variables used by the 
'run_analysis.R' script. 

See the codebook and READme files for the original data set at:
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones 

Also see the READme.txt file in this repository.

NOTES:

Names for variables listed above are derived from the 'features.txt' file in the 
original data set. Original data labels that contained characters invalid in 
column names have been replaced by the run_analysis.R script with '.'. Replaced
characters are:  '-', '(', and ')'.

Variable prefixes indicate units of measurement:
-- 't' indicates time measurements, in seconds
-- 'f' indicates frequencies, in Hz
